Proyectos

Análisis de Embudo de Conversión (Sales Funnel Analysis)

Objetivo:
Analizar el comportamiento de los usuarios a lo largo de un sales funnel para identificar en qué etapas del proceso se pierden potenciales clientes.

Problema:
En plataformas digitales es común que los usuarios abandonen el proceso de conversión en diferentes etapas, lo que dificulta identificar dónde se generan las principales pérdidas de oportunidades.

Qué se realizó:
Se llevó a cabo un Exploratory Data Analysis (EDA) sobre los datos de interacción de los usuarios, calculando tasas de conversión entre etapas del embudo para detectar posibles cuellos de botella.

Tecnologías utilizadas:

  • Python
  • Pandas
  • NumPy
  • Jupyter Notebook

Aprendizajes:
El análisis permitió identificar las etapas con menor conversión dentro del embudo, lo que puede ayudar a mejorar la experiencia de usuario y optimizar el proceso de adquisición de clientes.


Análisis de Datos de E-commerce

Objetivo:
Explorar datos de transacciones de una plataforma de comercio electrónico para comprender patrones de compra y variables que influyen en la generación de ingresos.

Problema:
Los negocios digitales generan grandes volúmenes de datos, pero convertir esa información en conocimiento útil requiere un proceso estructurado de análisis.

Qué se realizó:
Se analizó un conjunto de datos de transacciones mediante técnicas de Exploratory Data Analysis, identificando patrones de comportamiento de clientes y tendencias en las compras.

Tecnologías utilizadas:

  • Python
  • Pandas
  • Data Analysis
  • Data Visualization

Aprendizajes:
El análisis permitió comprender mejor el comportamiento de compra de los usuarios y detectar patrones relevantes en la generación de ingresos.
